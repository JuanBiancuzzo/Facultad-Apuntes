# Definición
---
Se establece como $$ M = MB + MI + MII $$ donde:
* $MB = 70W$ ([[Watt|watts]])

* $MI$ con la siguiente tabla
| Posición del cuerpo | Trabajo por segundo |
| ------------------- | ------------------- |
| Acostado o sentado  | 21                  |
| De pie              | 42                  |
| Caminando           | 140                 |
| Subiendo            | 210                    |

* MII con la siguiente tabla, 
| Tipo de trabajo         | Trabajo por segundo |
| ----------------------- | ------------------- |
| Manual ligero           | 28                  |
| Manual pesado           | 63                  |
| Con un brazo ligero     | 70                  |
| Con ambos brazos ligero | 189                 |
| Con el cuerpo pesado    | 490                    |
