# Definición
---
Cuando se realiza el experimento asociado a un [[Espacio de probabilidad|espacio de probabilidad]] $(\Omega,~ \mathcal{A},~ \mathbb{P})$ se observa alguno de los eventos elementales, por ejemplo $\tilde{\omega}$. Si sobre este espacio se define una [[Variable aleatoria|variable aleatoria]] $X$, al observar un evento elemental $\tilde{\omega}$, la variable aleatoria tomará el valor $X(\tilde{\omega}) = \tilde{x}$, y este valor particular se lo denomina realización de la variable aleatoria
