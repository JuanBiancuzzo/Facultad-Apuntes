# Definición
---
Se define una [[Función|función]] $f(t)$ como una función impar si $f(t) = -f(-t)$ 

Donde aparece la siguiente igualdad $$ \exists ~ g(t) : f(t) = \frac{g(t) - g(-t)}{2} $$
