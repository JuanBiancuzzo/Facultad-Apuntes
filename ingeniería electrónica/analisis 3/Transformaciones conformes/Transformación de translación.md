# Definición
---
Es una [[Transformación conforme]] donde se define como $T_p : \mathbb{C} \to \mathbb{C}$ tal que $$\begin{align} 
	T_p(z) = p + z && p \in \mathbb{C}
\end{align}$$
# Observación
---
* No es una [[Función lineal]]
