# Definición
---
Este [[Protocolo|protocolo]] de red, permite la comunicación entre [[redes/Redes de computadoras/Host.md|host]], en la [[Capa de Red|capa de red]]. Se ve separado en dos versiones
* [[Internet Protocol Versión 4|IPv4]]
* [[Internet Protocol Versión 6|IPv6]]
