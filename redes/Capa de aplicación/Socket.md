### Definición
---
Cualquier [[Paquete|mensaje]] enviado de un [[Proceso|proceso]] a otro tiene que ir por medio de una [[Red|red]]. Un proceso que envía un mensaje a, o recibe un mensaje de, la interfaz de red que se llama socket. También se refiere una como una [[Aplicación Programming Interface (API)|API]] a la interfaz entre una aplicación y la red
