# Definición
---
La frontera entre el [[Host|host]] y el enlace es conocida como interfaz. La frontera entre un [[Router|router]] y cualquier de sus links también se conoce como interfaz

Un router entonces puede tener múltiples, cada una con un link. El [[Internet Protocol Address|protocolo IP]] requiere que cada una de estas interfaces tenga su propia dirección
