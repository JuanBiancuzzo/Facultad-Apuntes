### Definición
---
El [[Protocolo|protocolo]] le asigna a cada nodo un código diferente, el cual utilizara para codificar los datos de los bits que enviará. Si estos códigos se seleccionan cuidadosamente, se puede permitir que múltiples nodos transmitan simultáneamente sin generar interferencia
