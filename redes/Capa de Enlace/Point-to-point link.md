### Definición
---
Un point-to-point link consiste en un único remitente en un extremo de [[Enlace|enlace]] y un único receptor en el otro extremo del enlace. Existe múltiples [[Protocolo|protocolos]] para el envío de [[Información|información]] a través de este enlace, entre ellos 
* [[Pont-to-point protocol (PPP)|Pont-to-point protocol (PPP)]]
* [[High-level data link control (HDLC)|High-level data link control (HDLC)]]
