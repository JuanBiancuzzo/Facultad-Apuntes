### Definición
---
Con lo que obtenemos del siguiente proceso:
![[Análisis de riesgo.canvas]]

Podemos completar la siguiente tabla para su análisis
| [[Riesgo]]                        | C   | P   | E   | GP  |
| --------------------------------- | --- | --- | --- | --- |
| Atrapamientos (mecánico)          |     |     |     |     |
| Caída de altura                   |     |     |     |     |
| [[Riesgo de incendio\|Incendios]] |     |     |     |     |
| [[Riesgo eléctrico\|Eléctrico]]   |     |     |     |     |
| $\vdots$                                  |     |     |     |     |

Donde $C$ son las consecuencia, $P$ la [[Probabilidad|probabilidad]] de ocurrencia y $E$ es la exposición (frecuencia con que se presenta la situación de riesgo), por último $GP$ se calcula como $$ GP = C \cdot P \cdot E $$
Se puede categorizar el $GP$ con la siguiente tabla
| Categoria | Descripción                                   | Mínimo | Máximo |
| --------- | --------------------------------------------- | ------ | ------ |
| Bajo      | Intervención a largo plazo o riesgo tolerable | 1      | 300    |
| Medio     | Intervención a corto plazo                    | 300    | 600    |
| Alto      | Intervención                                              |        |        |

Los valores de consecuencia, probabilidad y exposición las podemos encontrar en las siguientes tablas
| Valor | Consecuencias                                                                             |
| ----- | ----------------------------------------------------------------------------------------- |
| 10    | Muerte y/o daños mayores a 6000 dólares                                                   |
| 6     | [[Lesión\|Lesiones]] incapaces permanentes y/o daños entre 2000 y 6000 dólares            |
| 4     | [[Lesión\|Lesiones]] con incapacidad no permanentes y/o daños entre 600 y 2000 dólares    |
| 1     | [[Lesión\|Lesiones]] con heridas leves, contusiones, golpes y/o pequeños daños económicos |

| Valor | Probabilidad                                                                            |
| ----- | --------------------------------------------------------------------------------------- |
| 10    | Es el resultado más probable y esperado; si la situación de reisgo tiene lugar          |
| 7     | Es completamente posible, nada extraño. Tiene una probabilidad de ocurrencia del $50\%$ |
| 4     | Sería una rara coincidencia. Tiene una probabilidad de $20\%$                           |
| 1     | Nunca ha suedido en muchos años de exposición, el riesgo pero es concebible     |

| Valor | Exposición                                                        |
| ----- | ----------------------------------------------------------------- |
| 10    | La situación de riesgo ocurre continuamente o muchas veces al día |
| 6     | Frecuentemenete una vez al día                                    |
| 2     | Ocasionalmente o una vez por semana                               |
| 1     | Remotamente posible                                                                  |
