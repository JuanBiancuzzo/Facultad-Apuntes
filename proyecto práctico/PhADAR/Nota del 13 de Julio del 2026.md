# Progreso
---
Algo que quería agregar es que existe la idea de [[Radar de Apertura Sintética (Synthetic-aperture radar) (SAR))|SAR]] donde se aprovecha el movimiento a lo largo del espacio para recrear el uso de multiples antenas. Esto se podría aprovechar, conociendo que el sensor se puede estar moviendo, ya sea porque esta en un [[investigación/robótica/Robot|robot]] o por vibraciones, se podría intentar [[ingeniería en informática/proba/Inferencia estadística/Observable|estimar las diferencias de posición]], y también poder recibir información de la posición del sensor, como si información de un sensor se tratase. Todo esta información se puede utilizar como forma de mejorar la calidad de las observaciones, incluso mientra más movimiento es experimentado por el sensor
