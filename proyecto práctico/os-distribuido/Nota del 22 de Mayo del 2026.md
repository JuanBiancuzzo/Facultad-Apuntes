# Progreso
---
Algo interesante a considerar es que cambiar el [[investigación/ciencias de la computación/sistemas operativos/Sistema operativo|sistema operativo]] de un dispositivo conlleva muchos detalles que al empezar de $0$ es difícil de tener todo en cuenta, por eso estaría bueno la idea de pensar en crear una [[Aplicación|aplicación]] por encima de un sistema operativo preexistente como se hizo para el trabajo profesional ya mencionado, y al mismo tiempo para dispositivos que vaya creando, permitir esto a nivel de sistema operativo

Esta forma de plantearlo permite completamente extender más rápido la interoperabilidad del sistema, dando resultados apreciables desde el inicio. Esto daría una base en la cual extender y reemplazar otro sistema operativo, si esto se quisiese
