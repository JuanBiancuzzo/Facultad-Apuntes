# Progreso
---
Ayer y hoy, estuve haciendo que las referencias sean una nueva variable, creando un [[ingeniería en informática/taller/Sintaxis/Enum|enum]] en el caso de ser una referencia a multiples tablas

Ahora estoy definiendo como crear una view. Cada view esta definida por un nombre, un template, requisitos, y parámetros. Esto hace que el usuario defina completa como se ve la información

Para el siguiente paso, debería ver como hacer para definir endpoints de crear, de actualizar y de eliminar. Me imagino que deberían estar ligados a las tablas, pero tal vez se pueden vincular a las views para poder definir información que tengan estas
