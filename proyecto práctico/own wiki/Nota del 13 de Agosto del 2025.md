# Progreso
---
Pude finalmente hacer que dado un archivo [[Javascript Object Notation (JSON)|JSON]] se pueda generar las tablas, y toda la estructura que es necesaria para cargar la información. Además volví a cargar la información dado ese JSON por lo que volvemos a tener toda la información que teníamos cuando hice la página con [[HTMX|HTMX]]

Ahora estoy buscando como parsear el texto de [[investigación/ciencias de la computación/lenguaje de marcado/Markdown|Markdown]] y convertirlo en JSON. Mi intención es generar un [[ingeniería en informática/discreta/Grafos/Árbol|árbol]] del contenido, por lo tanto voy a generar mis estructuras necesarias para obtener toda la información de un [[ingeniería en informática/sisop/File system/Archivo|archivo]] markdown, y que esta estructura se pueda pasar a JSON

Algo que también me gustaría mencionar, es que la intención sería poder tener un editor de texto que me permita editar este contenido que estoy leyendo, por lo tanto, este árbol también lo vamos a usar para ejecutar modificaciones y ediciones cuando se quiera editar el texto
