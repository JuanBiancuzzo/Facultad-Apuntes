Sea $z \in \mathbb{C}$, tal que $z = x + i \cdot y$, se define como 
$$ arg : \mathbb{C} - \{0\} \to \mathbb{R} : arg(z) =
\begin{cases}
	tg^{-1}(\frac{y}{x}) & x > 0 \\
	tg^{-1}(\frac{y}{x}) + \pi & y \geq 0, x < 0 \\
	tg^{-1}(\frac{y}{x}) - \pi & y < 0, x < 0 \\
	\frac{\pi}{2} & y > 0, x = 0 \\
	-\frac{\pi}{2} & y < 0, x = 0 \\
\end{cases}
$$
