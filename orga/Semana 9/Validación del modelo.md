### Definición
---
En un [[Aprendizaje supervisado|aprendizaje supervisado]] se tiene que validar la respuesta de nuestro modelo, por lo que tendremos que dividir nuestro set de datos en dos:
* Training set
* Test set

El training set lo usaremos para entrenar, mientras que el test set lo usamos para medir la performance de nuestro modelo, lo cual implica el uso de alguna [[Métrica de un modelo|métrica]].
