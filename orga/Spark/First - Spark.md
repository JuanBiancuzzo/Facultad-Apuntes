### Definición
---
Es una [[Acción - Spark|acción]] que nos permite obtener el primer registros de un [[Resilent distributed dataset - Spark|RDD]].

``` python
rdd.first()
```

Es equivalente a un [[Take - Spark|take]] de un elemento.
