# Definición
---
Es equivalencia con el [[Rango de una variable aleatoria|rango]] pero generalizado para cualquier tipo de [[Variable aleatoria|variable aleatoria]]. Se define el soporte de una variable aleatoria como $$ S_x = \bigg\{ x \in \mathbb{R} : F_{x^+}(x) - F_{x^-}(x) \ne 0 \lor \frac{d}{dx}F_X(x) \ne 0 \bigg\} $$
