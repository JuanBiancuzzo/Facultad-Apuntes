# Definición
---
Dado un [[Vector aleatorio|vector aleatorio]] $\mathbb{X} = (X_1, X_2, \cdots, X_n)$ y una [[Función de probabilidad conjunta|función de probabilidad conjunta]] $p_{X_1, X_2, ..., X_n}$, se define el rango de $\mathbb{X}$ como $$ R_\mathbb{X} = \{ (x_1, x_2, \cdots, x_n) \in \mathbb{R}^n : p_{X_1, X_2, ..., x_n}(x_1, x_2, \cdots, x_n) > 0 \} $$
