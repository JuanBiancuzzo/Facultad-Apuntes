# Definición
---
Dado una [[Variable aleatoria|variable aleatoria]] $X$ y una [[Función de probabilidad|función de probabilidad]] $p_X$, se define el rango de $X$ como $$ R_X = \{ x \in \mathbb{R} : p_X(x) > 0 \} $$
