# Definición
---
Corresponde a cada fila que compone la [[Tabla SQL|tabla]]. Ahí se componen los datos y los registros. Eventualmente pueden ser nulos en su almacenamiento

## Ejemplo
---
Teniendo la tabla

| index | nombre | num_compras |
| ----- | ------ | ----------- |
| ...   | ...    | ...         |
| 34    | Juan   | 0           |
| 35    | Lucia  | 3           |
| 36    | Sofia  | 4           |
| ...   | ...    | ...         |

Y un posible registro sería

| index | nombre |
| ----- | ------ |
| 34    | Juan   |
