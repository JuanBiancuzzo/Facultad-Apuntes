# Definición
---
Esta [[Estructura de control selectiva|estructura de control selectiva]] alterar el flujo normal del [[ingeniería en informática/sisop/La abstracción de proceso/Programa|programa]] según se cumpla o no una determinada condición, se la denomina if múltiple o if anidado, esta dada por

```c
if (expresion1) 
	accion1;
else if (expresion2)
	accion2;
else if (expresion3)
	accion3;
else 
	accion4;
```
