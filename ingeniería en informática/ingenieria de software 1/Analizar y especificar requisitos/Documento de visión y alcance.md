# Definición
---
Desarrollado en las primera fases/iteraciones. Se define el [[Sistema]] a desarrollar desde el punto de vista de las partes [[ingeniería en informática/ingenieria de software 1/Descubrimiento de requisitos/Interesados en el producto|interesadas]], donde esta es la base para [[ingeniería en informática/ingenieria de software 1/Analizar y especificar requisitos/Especificación de requisitos|especificaciones]] más detalladas.

Puede incluir una definición del problema a resolver o de la oportunidad de negocio a aprovechar, una descripción de los usuarios e interesados, una descripción de las prestaciones del producto/sistema a desarrollar, [[Riesgo|riesgos]], valor para el negocio, supuestos, restricciones, etc.
