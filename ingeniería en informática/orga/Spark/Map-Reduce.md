# Definición
---
Es un procesamiento distribuido de datos utilizando un [[Cluster]] de computadoras. Para esto se necesita un [[Almacenamiento distribuido]] que se encargue de guardar la información para procesar, y un [[File system distribuido|file system distribuido]] para organizarlo. Esto lo aplica en dos etapas
* [[Map#En Spark|Map]]
* [[Reduce|Reduce]]
