# Definición
---
El Radio Network Controller (RNC) típicamente controla múltiples [[Base Transceiver Station|BTS]], similar a la [[Base Station Controller|BSC]] a las que vimos en las redes 2G. Un RNC conecta tanto circuit-switched cellular voice network a través de [[Mobile Switching Center|MSC]], y packet-switched internet a través de [[Serving General packet radio service Support Nodes|SGSN]]. Mientras que los servicios de voz y de datos usan redes centrales distintas, comparten la radio access network y el RNC

Un cambio significativo entre las redes 3G UMTS y las redes 2G es que en lugar de utilizar el esquema de GMS [[Frequency Division Multiplexing|FDMA]]/[[Time Division Multiplexing|TDMA]], utiliza una técnica de [[Code Division Multiple Access|CDMA]] conocida como [[Direct Sequence Wideband Code Division Multiple Access (Direct Sequence Wideband CDMA) (DW-WCDMA)|Direct Sequence Wideband CDMA (DW-WCDMA)]], dentro de slots de TDMA
