# Definición
---
El throughput en cualquier instante del tiempo es la tasa a la cual el receptor está recibiendo el [[Paquete|paquete]]

Supongamos [[Arquitectura cliente servidor|un servidor y un cliente]] conectados por dos links y un [[Router|router]]. Si definimos $R_s$ como la tasa del link entre el servidor y el router, y $R_c$ como la tasa de link entre el router y el cliente. Entonces el throughput será el [[Mínimo|mínimo]] de estos dos valores, este será el cuello de botella del link
