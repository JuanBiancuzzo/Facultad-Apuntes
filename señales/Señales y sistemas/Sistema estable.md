### Definición
---
Existen varios criterios de estabilidad. Se va a tomar que un sistema es estable si para entradas acotadas la salida pertenece acotada. Este criterio de estabilidad se conoce comúnmente como [[Bounded Input-Bounded Output (BIBO)|BIBO (Bounded Input-Bounded Output)]] ![[Bounded Input-Bounded Output (BIBO)#Definición]]
