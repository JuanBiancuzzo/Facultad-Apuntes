# Enunciado
---
El error de un [[Global Positioning System (GPS)|GPS]] en una determinada dirección posee una [[colección/distribuciones/continua/Distribución Normal|distribución normal]] de [[ingeniería en informática/proba/Representación de variables aleatorias/Esperanza|media]] nula y [[ingeniería en informática/proba/Representación de variables aleatorias/Varianza|varianza]] $\frac{1}{\tau}$, donde representa la precisión del dispositivo. A priori la precisión $\tau \sim \text{Exp}(1)$ y después de $10$ mediciones se obtuvo $\sum_{i = 1}^{10} x_i^{10} = 10$

1. Obtener la distribución a posteriori ^punto-1
2. Obtener la [[ingeniería electrónica/taller de señales/Repaso de probabilidad y estadística/Estadística Bayesiana#^distribucion-predictiva|distribución predictiva]] ^punto-2
