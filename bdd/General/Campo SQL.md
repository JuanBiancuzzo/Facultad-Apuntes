### Definición
---
Corresponde al nombre de la columna. Debe ser único y además de tener un tipo de dato asociado

#### Ejemplo
---
Teniendo la tabla

| index | nombre | num_compras |
| ----- | ------ | ----------- |
| ...   | ...    | ...         |
| 34    | Juan   | 0           |
| 35    | Lucia  | 3           |
| 36    | Sofia  | 4           |
| ...   | ...    | ...         |

Y una posible campo sería

| nombre |
| ------ |
| ...    |
| Juan   |
| Lucia  |
| Sofia  |
| ...    |
