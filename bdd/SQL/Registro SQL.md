### Definición
---
Los registro son los valores de un conjunto de las columnas de una [[Tabla SQL|tabla]]. 

#### Ejemplo
---
Teniendo la tabla

| index | nombre | num_compras |
| ----- | ------ | ----------- |
| ...   | ...    | ...         |
| 34    | Juan   | 0           |
| 35    | Lucia  | 3           |
| 36    | Sofia  | 4           |
| ...   | ...    | ...         |

Y un posible registro sería

| index | nombre |
| ----- | ------ |
| 34    | Juan   |
