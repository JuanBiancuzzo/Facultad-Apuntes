# Definición
---
Cubos de marcha es un [[Algoritmo|algoritmo]] de gráficos por [[Computadora|computadora]] publicado en las memorias del congreso [[SIGGRAPH|SIGGRAPH]] en 1987 por Lorensen y Cline.​ Este algoritmo tiene como objetivo extraer una malla poligonal de una isosuperficie de un campo escalar discreto tridimensional (a veces llamado vóxeles). Este artículo es uno de los más citados en el campo de gráficos por computadora. Las aplicaciones de este algoritmo se refieren principalmente a visualizaciones médicas como TAC e imágenes de datos de escáner de IRM, y efectos especiales o modelación 3-D donde normalmente son llamados [[Metaball|metaballs]] o metasuperficies. Un método bidimensional análogo es llamado algoritmo cuadrados de marcha (marching squares)<sup><a href="#ref-284" style="color: inherit; text-decoration: none;">[284]</a></sup>
