# Descripción
---
Este curso ofrece una introducción a la modelización matemática de problemas computacionales. Cubre los algoritmos comunes, los paradigmas algorítmicos y las estructuras de datos utilizadas para resolver estos problemas. El curso enfatiza la relación entre los algoritmos y la programación, e introduce medidas básicas de rendimiento y técnicas de análisis para estos problemas
