# Descripción
---
El curso que ofrece una introducción práctica basada en proyectos para construir sistemas de software escalables y de alto rendimiento. Los temas incluyen análisis de rendimiento, técnicas algorítmicas para alto rendimiento, optimizaciones a nivel de instrucciones, optimizaciones de caché, programación paralela y construcción de sistemas escalables. El lenguaje de programación del curso es C
