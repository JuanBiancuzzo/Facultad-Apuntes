# Descripción
---
El curso Construcción de Software introduce principios fundamentales y técnicas de desarrollo de software, es decir, cómo escribir software que esté libre de errores, sea fácil de entender y listo para cambios. El curso incluye conjuntos de problemas y un proyecto final. Los temas importantes incluyen especificaciones e invariantes; pruebas; tipos de datos abstractos; patrones de diseño para la programación orientada a objetos; programación concurrente y concurrencia; y programación funcional
